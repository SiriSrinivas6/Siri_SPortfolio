Tableau Projects

This holds my Tableau Dashboards.

Tableau Public is a free data visualization tool for creating interactive charts and dashboards. Users can connect to various data sources to create visually engaging visualizations. Public dashboards can be shared online via the Tableau Public website. They are commonly used for storytelling and data journalism. Dashboards allow for interactive exploration of data through filtering and sorting. Users should be cautious about sharing sensitive data as Tableau Public dashboards are publicly accessible. Tableau Public has some limitations compared to Tableau Desktop or Online. Despite limitations, it remains a powerful tool for sharing insights with a broader audience. It enables users to convey complex information in an accessible manner. Overall, Tableau Public is a valuable resource for creating and sharing interactive data visualizations.
